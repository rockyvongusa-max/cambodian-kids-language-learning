# Cambodian Kids Language Learning Website

Help Cambodian kids learn English and Chinese through interactive visual lessons.

## Features
- K-6 grade levels
- Yellow duck mascot
- Claymorphism UI
- Daily conversation practice
- Speech recognition
- Offline support

## Tech Stack
- Frontend: React + Tailwind CSS
- Backend: Node.js
- Database: PostgreSQL
- PWA enabled
